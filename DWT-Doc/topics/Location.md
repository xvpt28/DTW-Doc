# Location

## Description

Store All the location information (e.g. Level 3 warehouse, Level 4 warehouse, etc.)

## Database

### Table

`required`

### Columns

| Name | Type   | Description                  | Foreign Key |
|------|--------|------------------------------|-------------|
| id   | UUID   | Primary Key of the operation | No          |
| name | string | Name of the location         | No          |
